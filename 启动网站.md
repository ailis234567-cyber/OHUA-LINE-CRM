# 🌐 启动 Web 网站指南

## 步骤 1: 安装 Flask（如果还没安装）

```bash
cd /Users/huihui/live-monitor
source venv/bin/activate
pip install flask
```

## 步骤 2: 测试是否能正常启动

```bash
python test_web.py
```

这会检查所有依赖和文件是否正常。

## 步骤 3: 启动 Web 服务器

### 方法 1: 使用启动脚本
```bash
./start_web.sh
```

### 方法 2: 直接运行
```bash
python web_app.py
```

## 步骤 4: 访问网站

启动成功后，你会看到：
```
🌐 直播监控截图 Web 应用
==================================================
📂 截图目录: /Users/huihui/live-monitor/screenshots
🚀 启动服务器...
📍 访问地址: http://localhost:5000
==================================================
 * Running on http://0.0.0.0:5000
```

然后在浏览器中打开：
- **本地访问**: http://localhost:5000
- **局域网访问**: http://你的IP地址:5000

## ❌ 常见问题

### 1. 提示 "Flask 未安装"
```bash
source venv/bin/activate
pip install flask
```

### 2. 提示 "端口被占用"
修改 `web_app.py` 最后一行，换个端口：
```python
app.run(host='0.0.0.0', port=5001, debug=True)  # 改为5001
```

### 3. 无法访问网站
- 检查防火墙设置
- 确认服务器已启动（看到 "Running on" 消息）
- 尝试使用 `127.0.0.1:5000` 而不是 `localhost:5000`

### 4. 图片无法显示
- 确认 `screenshots` 目录下有图片
- 检查 `config.yaml` 中的 `save_dir` 配置是否正确

### 5. 模板文件找不到
确认以下文件存在：
- `templates/index.html`
- `static/css/style.css`
