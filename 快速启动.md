# 🚀 快速启动指南

## 第 1 步：配置 ngrok

运行配置脚本：
```bash
./配置ngrok.sh
```

或者手动配置：
```bash
ngrok config add-authtoken 36ThduP2qwzn7vHLfg87kCYQnkG_6agLLUmUz2f9u7E1d773K
```

## 第 2 步：启动服务（需要两个终端）

### 终端 1 - 启动 Web 应用
```bash
cd /Users/huihui/live-monitor
source venv/bin/activate
python web_app.py
```

等待看到：
```
📍 访问地址: http://localhost:5001
 * Running on http://0.0.0.0:5001
```

### 终端 2 - 启动 ngrok
```bash
ngrok http 5001
```

你会看到类似：
```
Forwarding  https://abc123.ngrok-free.app -> http://localhost:5001
```

## 第 3 步：访问网站

复制 ngrok 显示的网址（如：`https://abc123.ngrok-free.app`），在浏览器中打开即可！

---

## 📝 注意事项

1. **保持两个终端都运行**：Web 应用和 ngrok 都必须保持运行
2. **网址会变化**：免费版每次启动网址可能不同
3. **停止服务**：按 `Ctrl+C` 停止，停止后网址失效
